# VK Group Auto Posting with n8n

Автоматизация публикации контента в группу ВКонтакте с использованием n8n.

## Возможности

- ✅ Автоматическая публикация текстовых постов
- ✅ Публикация постов с изображениями
- ✅ Создание и публикация опросов (polls)
- ✅ Планирование публикаций по расписанию
- ✅ Управление контентом через Google Sheets
- ✅ Автоматическое отслеживание опубликованных постов

## Архитектура

Workflow состоит из следующих компонентов:

1. **Schedule Trigger** - запускает workflow по расписанию (по умолчанию каждый час)
2. **Google Sheets Integration** - читает данные о постах из таблицы
3. **Filter** - отбирает посты, готовые к публикации
4. **Conditional Logic** - определяет тип контента (текст/изображение/опрос)
5. **VK API Integration** - публикует контент в группу ВКонтакте
6. **Status Update** - обновляет статус публикации в Google Sheets

## Требования

### 1. n8n Installation

Установите n8n одним из способов:

```bash
# NPM
npm install n8n -g

# Docker
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n

# Docker Compose (рекомендуется)
# См. docker-compose.yml в репозитории
```

### 2. VK API Access Token

Для работы с VK API необходимо получить access token:

1. Перейдите на https://vk.com/apps?act=manage
2. Создайте Standalone приложение
3. Получите access token с правами:
   - `photos` - для загрузки изображений
   - `wall` - для публикации на стене
   - `polls` - для создания опросов

**Важно:** Используйте Service Access Token для работы от имени сообщества.

#### Получение Service Access Token:

1. Откройте настройки вашей группы ВКонтакте
2. Перейдите в раздел "Работа с API"
3. Создайте ключ доступа с правами:
   - Управление сообществом
   - Фотографии
   - Создание опросов
4. Скопируйте полученный токен

### 3. Google Sheets Setup

Создайте Google Sheets таблицу со следующей структурой:

| post_date | post_type | content | image_url | poll_question | poll_options | published | published_at |
|-----------|-----------|---------|-----------|---------------|--------------|-----------|--------------|
| 2026-02-10 12:00:00 | text | Привет, мир! | | | | false | |
| 2026-02-10 14:00:00 | image | Посмотрите на это фото! | https://example.com/image.jpg | | | false | |
| 2026-02-10 16:00:00 | poll | | | Какой ваш любимый цвет? | Красный\|Синий\|Зеленый | false | |

**Описание колонок:**

- `post_date` - дата и время публикации (формат: YYYY-MM-DD HH:MM:SS)
- `post_type` - тип поста: `text`, `image`, или `poll`
- `content` - текст поста
- `image_url` - URL изображения (для типа `image`)
- `poll_question` - вопрос опроса (для типа `poll`)
- `poll_options` - варианты ответов через разделитель `|` (для типа `poll`)
- `published` - статус публикации: `false` или `true`
- `published_at` - дата и время фактической публикации

## Установка и настройка

### Шаг 1: Импорт workflow в n8n

1. Откройте n8n в браузере (обычно http://localhost:5678)
2. Нажмите на "+" для создания нового workflow
3. Нажмите на меню (три точки) → "Import from File"
4. Выберите файл `vk-n8n-workflow.json`

### Шаг 2: Настройка VK API Credentials

1. В n8n перейдите в "Credentials" → "New"
2. Выберите "HTTP Query Auth"
3. Создайте credential с именем "VK API Credentials"
4. Добавьте следующие параметры:
   - Name: `access_token`
   - Value: ваш VK access token
5. Добавьте custom field:
   - Name: `vkGroupId`
   - Value: ID вашей группы (отрицательное число, например: `-123456789`)

**Как узнать ID группы:**
- Откройте вашу группу в VK
- ID находится в URL: `vk.com/club123456789` → ID = `-123456789`
- Или используйте API метод: https://vk.com/dev/groups.getById

### Шаг 3: Настройка Google Sheets

1. В n8n перейдите в "Credentials" → "New"
2. Выберите "Google Sheets OAuth2 API"
3. Следуйте инструкциям для подключения вашего Google аккаунта
4. В узле "Read Posts from Google Sheets":
   - Укажите ID вашей таблицы (из URL таблицы)
   - Выберите нужный лист (Sheet1)

### Шаг 4: Настройка расписания

В узле "Schedule Trigger" настройте частоту проверки:

- **Каждый час** (по умолчанию): проверяет новые посты каждый час
- **Каждые 30 минут**: измените interval на 30 минут
- **Конкретное время**: используйте Cron expression

Примеры Cron выражений:
```
0 9,12,15,18 * * * - в 9:00, 12:00, 15:00, 18:00
0 */2 * * * - каждые 2 часа
0 10 * * 1-5 - в 10:00 по будням
```

### Шаг 5: Активация workflow

1. Проверьте все настройки
2. Нажмите "Execute Workflow" для тестового запуска
3. Если все работает корректно, активируйте workflow переключателем "Active"

## Использование

### Публикация текстового поста

Добавьте строку в Google Sheets:

```
post_date: 2026-02-10 12:00:00
post_type: text
content: Привет! Это автоматический пост из n8n 🚀
published: false
```

### Публикация поста с изображением

```
post_date: 2026-02-10 14:00:00
post_type: image
content: Посмотрите на это красивое фото!
image_url: https://example.com/photo.jpg
published: false
```

**Требования к изображениям:**
- Формат: JPG, PNG, GIF
- Максимальный размер: 50 МБ
- URL должен быть публично доступен

### Публикация опроса

```
post_date: 2026-02-10 16:00:00
post_type: poll
content: Пройдите наш опрос!
poll_question: Какой язык программирования вы предпочитаете?
poll_options: Python|JavaScript|Go|Rust
published: false
```

**Требования к опросам:**
- Минимум 2 варианта ответа
- Максимум 10 вариантов ответа
- Варианты разделяются символом `|`

## Мониторинг и отладка

### Просмотр логов выполнения

1. В n8n откройте вкладку "Executions"
2. Выберите нужное выполнение
3. Просмотрите данные на каждом узле

### Типичные ошибки

**Ошибка: "Invalid access token"**
- Проверьте правильность access token
- Убедитесь, что токен не истек
- Проверьте права доступа токена

**Ошибка: "Access denied"**
- Убедитесь, что используете Service Access Token
- Проверьте права токена в настройках группы

**Ошибка: "Photo upload failed"**
- Проверьте доступность URL изображения
- Убедитесь, что формат изображения поддерживается
- Проверьте размер файла

**Ошибка: "Poll creation failed"**
- Проверьте количество вариантов ответа (2-10)
- Убедитесь, что варианты разделены символом `|`

## Расширенные возможности

### Отложенная публикация

Workflow автоматически публикует посты, когда `post_date` меньше или равна текущему времени. Вы можете планировать публикации на дни и недели вперед.

### Массовая загрузка контента

Подготовьте несколько постов в Google Sheets заранее, и они будут автоматически опубликованы в назначенное время.

### Интеграция с другими сервисами

Workflow можно расширить, добавив:
- Webhook trigger для публикации по запросу
- RSS feed reader для автоматического импорта контента
- AI nodes для генерации контента
- Telegram bot для управления публикациями

### Пример расширения с RSS

Добавьте узел "RSS Feed Read" перед Google Sheets для автоматического импорта новостей:

```json
{
  "node": "RSS Feed Read",
  "type": "n8n-nodes-base.rssFeedRead",
  "parameters": {
    "url": "https://example.com/feed.xml"
  }
}
```

## Безопасность

### Рекомендации:

1. **Не публикуйте access token в открытом доступе**
2. **Используйте переменные окружения** для хранения токенов
3. **Ограничьте права токена** только необходимыми
4. **Регулярно обновляйте токены**
5. **Используйте HTTPS** для всех внешних запросов
6. **Ограничьте доступ к Google Sheets** только для n8n аккаунта

### Переменные окружения

Для production использования рекомендуется хранить credentials в переменных окружения:

```bash
# .env файл
VK_ACCESS_TOKEN=your_token_here
VK_GROUP_ID=-123456789
GOOGLE_SHEETS_FILE_ID=your_sheet_id
```

## Troubleshooting

### Workflow не запускается по расписанию

1. Убедитесь, что workflow активирован (переключатель "Active")
2. Проверьте настройки Schedule Trigger
3. Проверьте логи n8n

### Посты не публикуются

1. Проверьте фильтр "Filter Posts Ready to Publish"
2. Убедитесь, что `post_date` <= текущее время
3. Проверьте, что `published` = `false`
4. Проверьте формат даты в Google Sheets

### Изображения не загружаются

1. Проверьте доступность URL (откройте в браузере)
2. Убедитесь, что сервер поддерживает прямые ссылки
3. Проверьте размер изображения
4. Попробуйте другой формат (JPG вместо PNG)
