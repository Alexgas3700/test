# Cursor Test - Launch Without Draft Field

## Описание

Этот проект демонстрирует успешный запуск Cursor без использования поля `draft` в конфигурации.

## Структура проекта

- `cursor-test.json` - Основной конфигурационный файл без поля draft
- `test` - Тестовый файл
- `README.md` - Документация

## Конфигурация

Конфигурационный файл `cursor-test.json` содержит все необходимые поля для работы, но намеренно **не содержит** поле `draft`, чтобы подтвердить, что система работает корректно без него.

## Результат

✅ Система успешно запускается и работает без поля `draft` в конфигурации.
