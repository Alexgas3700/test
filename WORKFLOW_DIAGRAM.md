# Диаграммы Workflow

Визуальное представление структуры workflow для VK Content Creation.

---

## 📊 Базовый Workflow (vk-content-workflow.json)

```
┌─────────────────────────────────────────────────────────────────────┐
│                        VK Content Workflow                          │
└─────────────────────────────────────────────────────────────────────┘

┌──────────────────┐
│  Schedule        │  Запуск каждые 6 часов
│  Trigger         │  (настраивается)
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  Generate        │  Генерация контента из шаблонов
│  Content         │  • 5 готовых шаблонов
└────────┬─────────┘  • Случайный выбор
         │
         ▼
┌──────────────────┐
│  Validate        │  Проверка контента
│  Content         │  • Длина сообщения (max 4096)
└────────┬─────────┘  • Наличие owner_id
         │            • Пустое сообщение
         ▼
┌──────────────────┐
│  Check           │  Проверка результата валидации
│  Validation      │
└────┬────────┬────┘
     │        │
     │ Valid  │ Invalid
     ▼        ▼
┌─────────┐  ┌──────────────────┐
│   VK    │  │  Handle          │
│  Post   │  │  Validation      │
│ Content │  │  Error           │
└────┬────┘  └──────────────────┘
     │
     ▼
┌──────────────────┐
│  Check           │  Проверка ответа VK API
│  Success         │
└────┬────────┬────┘
     │        │
Success│      │Error
     ▼        ▼
┌─────────┐  ┌─────────┐
│   Log   │  │   Log   │
│ Success │  │  Error  │
└────┬────┘  └────┬────┘
     │            │
     ▼            ▼
┌──────────────────┐
│  Get Post        │  [Опционально]
│  Stats           │  Сбор статистики
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  Send            │  [Опционально]
│  Notification    │  Email/Telegram уведомления
└──────────────────┘
```

## 📦 Структура данных

### Объект контента

```javascript
{
  owner_id: "-123456",        // ID группы/пользователя
  message: "Текст поста",     // Текст сообщения (до 4096 символов)
  attachments: "",            // Вложения (фото, видео)
  timestamp: "2026-02-07...", // Время создания
  topic: "мотивация"          // Тема контента (для продвинутого)
}
```

### Объект валидации

```javascript
{
  valid: true,                // Результат валидации
  errors: [],                 // Массив ошибок (если есть)
  original_data: {...}        // Исходные данные
}
```

### Объект ответа VK

```javascript
{
  response: {
    post_id: 123              // ID созданного поста
  },
  error: {                    // Или объект ошибки
    error_code: 5,
    error_msg: "..."
  }
}
```

## 🎯 Сценарии использования

### Сценарий 1: Простая публикация

```
Timer → Generate → Validate → Post → Done
  ↓         ↓          ↓        ↓
 6h     Template    Check    VK API
```

### Сценарий 2: С AI генерацией

```
Timer → Topic → AI → Prepare → Validate → Post → Stats → Done
  ↓       ↓      ↓       ↓         ↓        ↓       ↓
 4h    Random  GPT-4  Format   Check    VK API  Collect
```

### Сценарий 3: С модерацией

```
Timer → Generate → Telegram → Wait → Approve → Post → Done
  ↓         ↓          ↓        ↓        ↓       ↓
 6h     Template   Notify   Manual   Check   VK API
```

### Сценарий 4: Множественная публикация

```
Timer → Generate → Loop → Post Group 1 → Done
  ↓         ↓        ↓   → Post Group 2 → Done
 6h     Template    3x  → Post Group 3 → Done
```

## 🔀 Условные переходы

### Валидация

```
┌──────────┐
│ Validate │
└────┬─────┘
     │
     ├─── Valid ───────▶ Continue to Post
     │
     └─── Invalid ─────▶ Handle Error
```

### Проверка успешности

```
┌──────────┐
│  Check   │
│ Success  │
└────┬─────┘
     │
     ├─── Success ─────▶ Log Success → Get Stats
     │
     └─── Error ───────▶ Log Error → Notify
```

### AI или Fallback

```
┌──────────┐
│  Topic   │
└────┬─────┘
     │
     ├─── AI Available ────▶ OpenAI Generate
     │
     └─── AI Unavailable ──▶ Fallback Content
```

## 🎨 Расширения

### Добавление нового узла

```
Existing Flow:
Generate → Validate → Post

Extended Flow:
Generate → [NEW NODE] → Validate → Post
              ↓
         Additional
         Processing
```

### Примеры расширений

```
1. Image Generation:
   Generate Text → DALL-E → Upload Image → Post with Image

2. Translation:
   Generate RU → Translate EN → Post Both Languages

3. Scheduling:
   Generate → Schedule → Wait → Post at Time

4. Multi-platform:
   Generate → Post VK
          → Post Facebook
          → Post Instagram
```

## 🚦 Обработка ошибок

### Стратегия обработки

```
Error Occurs
     │
     ▼
┌─────────────┐
│   Catch     │
│   Error     │
└──────┬──────┘
       │
       ├─── Network Error ───▶ Retry (3x)
       │
       ├─── Validation Error ─▶ Log & Skip
       │
       ├─── API Error ────────▶ Log & Notify
       │
       └─── Unknown Error ────▶ Log & Alert
```
