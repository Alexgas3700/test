### Технические файлы

#### 🔄 email-campaign-workflow.json (15KB)
Полнофункциональный n8n workflow с 14 узлами:

1. **Manual Trigger** - запуск workflow
2. **Get Email List** - получение контактов из Google Sheets
3. **Filter Active Contacts** - фильтрация активных контактов
4. **Personalize Emails** - персонализация писем
5. **Split In Batches** - разделение на партии
6. **Wait Between Batches** - пауза между партиями
7. **Send Email** - отправка письма
8. **Log Success** - логирование успеха
9. **Log Error** - логирование ошибок
10. **Update Google Sheet** - обновление таблицы
11. **Log Error to Sheet** - запись ошибок в таблицу
12. **No Active Contacts** - обработка отсутствия контактов
13. **Campaign Summary** - сводка по кампании
14. **Send Report** - отправка отчета администратору

**Особенности:**
- Полная обработка ошибок
- Автоматическое логирование
- Batch processing
- Персонализация
- Отчетность

#### 🎨 email-templates.md (25KB)
4 готовых HTML-шаблона:

1. **Промо-акция** - с градиентами и современным дизайном
2. **Новостная рассылка** - для регулярных обновлений
3. **Образовательный контент** - с пошаговыми инструкциями
4. **Приветственное письмо** - для новых подписчиков

**Каждый шаблон включает:**
