# Быстрый старт - VK Post Workflow

Начните публиковать посты в VK через n8n за 10 минут!

## Что вам понадобится

- ✅ Установленный n8n
- ✅ Аккаунт VK
- ✅ Группа VK (где вы администратор)

## Шаг 1: Создайте VK приложение (5 минут)

### 1.1. Перейдите на VK Developers

Откройте [https://dev.vk.com/](https://dev.vk.com/) и войдите в свой аккаунт.

### 1.2. Создайте приложение

1. Нажмите **"Создать приложение"**
2. Название: `n8n Integration`
3. Платформа: **"Standalone приложение"**
4. Нажмите **"Создать"**

### 1.3. Сохраните данные

Скопируйте и сохраните:
- **ID приложения** (например: 12345678)
- **Защищенный ключ** (например: aBcDeFgH...)

## Шаг 2: Получите токен доступа (2 минуты)

### 2.1. Откройте URL авторизации

Замените `YOUR_APP_ID` на ваш ID приложения и откройте в браузере:

```
https://oauth.vk.com/authorize?client_id=YOUR_APP_ID&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=photos,wall,groups,offline&response_type=token&v=5.131
```

### 2.2. Разрешите доступ

Нажмите **"Разрешить"** на странице авторизации.

### 2.3. Скопируйте токен

После авторизации вы увидите URL вида:
```
https://oauth.vk.com/blank.html#access_token=YOUR_TOKEN_HERE&expires_in=0&user_id=123456
```

Скопируйте значение `access_token` (это ваш токен).

## Шаг 3: Узнайте ID группы (1 минута)

### Вариант 1: Из URL

Если URL вашей группы: `https://vk.com/club123456789`
- Ваш Group ID: `123456789`

### Вариант 2: Через API

Замените `YOUR_TOKEN` и `mygroup` на ваши данные:

```bash
curl "https://api.vk.com/method/groups.getById?group_id=mygroup&access_token=YOUR_TOKEN&v=5.131"
```

В ответе найдите поле `"id"`.

## Шаг 4: Импортируйте workflow в n8n (1 минута)

### 4.1. Скачайте workflow

Скачайте файл `vk-post-workflow.json` из этого репозитория.

### 4.2. Импортируйте в n8n

1. Откройте n8n
2. Нажмите на меню (☰) → **"Import from File"**
3. Выберите файл `vk-post-workflow.json`
4. Нажмите **"Import"**

## Шаг 5: Настройте credentials (1 минута)

### 5.1. Создайте VK credential

1. В n8n откройте **Credentials** (иконка ключа)
2. Нажмите **"+ New"**
3. Выберите **"VK API"**
4. Вставьте ваш токен доступа
5. Название: `VK OAuth2 account`
6. Нажмите **"Save"**

### 5.2. Привяжите credential к узлам

1. Откройте импортированный workflow
2. Кликните на узел **"Upload Photo to VK"**
3. В поле **"Credential to connect with"** выберите `VK OAuth2 account`
4. Повторите для узла **"Create VK Post"**
5. Нажмите **"Save"** в правом верхнем углу

## Шаг 6: Настройте данные поста (1 минута)

### 6.1. Откройте узел "Set Post Data"

Кликните на узел **"Set Post Data"** в workflow.

### 6.2. Измените значения

Замените значения на свои:

```json
{
  "postText": "Привет! Это мой первый автоматический пост через n8n! 🚀",
  "imageUrl": "https://picsum.photos/800/600",
  "groupId": "123456789"
}
```

**Важно:** Замените `123456789` на ID вашей группы!

### 6.3. Сохраните

Нажмите **"Save"** в узле.

## Шаг 7: Запустите workflow! (10 секунд)

### 7.1. Выполните workflow

Нажмите кнопку **"Execute Workflow"** в правом верхнем углу.

### 7.2. Проверьте результат

1. Дождитесь завершения выполнения (все узлы станут зелеными)
2. Откройте вашу группу VK
3. Вы должны увидеть новый пост!

## 🎉 Готово!

Поздравляем! Вы успешно настроили автоматическую публикацию в VK через n8n.

## Что дальше?

### Автоматизируйте публикацию

Замените **Manual Trigger** на:

#### Публикация по расписанию

1. Удалите узел **"When clicking 'Test workflow'"**
2. Добавьте узел **"Schedule Trigger"**
3. Настройте расписание (например, каждый день в 9:00):
   ```
   Cron Expression: 0 9 * * *
   ```

#### Публикация через webhook

1. Удалите узел **"When clicking 'Test workflow'"**
2. Добавьте узел **"Webhook"**
3. Настройте:
   - HTTP Method: `POST`
   - Path: `vk-post`
4. Отправляйте POST запросы на URL webhook

### Публикуйте несколько изображений

Используйте workflow `vk-post-multiple-images-workflow.json`:

```json
{
  "postText": "Смотрите эти фото!",
  "imageUrls": [
    "https://picsum.photos/800/600?random=1",
    "https://picsum.photos/800/600?random=2",
    "https://picsum.photos/800/600?random=3"
  ],
  "groupId": "123456789"
}
```

### Интегрируйте с другими сервисами

- **Google Sheets** - читайте посты из таблицы
- **Telegram** - пересылайте посты из Telegram в VK
- **RSS** - автоматически публикуйте новости из RSS
- **OpenAI** - генерируйте тексты постов с помощью AI

## Частые вопросы

### Как публиковать только текст без изображения?

1. Удалите узлы **"Download Image"** и **"Upload Photo to VK"**
2. Соедините **"Set Post Data"** напрямую с **"Create VK Post"**
3. В узле **"Create VK Post"** удалите поле **"Attachments"**

### Как изменить текст поста?

В узле **"Set Post Data"** измените значение `postText`:

```json
{
  "postText": "Ваш новый текст поста"
}
```

### Как публиковать в несколько групп?

1. Добавьте узел **"Split In Batches"** после **"Set Post Data"**
2. Передайте массив Group ID
3. Workflow будет выполняться для каждой группы

### Ошибка "Invalid access token"

1. Проверьте, что токен скопирован полностью
2. Получите новый токен (см. Шаг 2)
3. Обновите credential в n8n

### Ошибка "Access denied"

1. Убедитесь, что вы администратор группы
2. Проверьте, что токен имеет права `photos`, `wall`, `groups`
3. Получите токен заново с правильными правами

### Изображение не загружается

1. Проверьте, что URL изображения доступен (откройте в браузере)
2. Убедитесь, что формат изображения поддерживается (JPG, PNG, GIF)
3. Проверьте размер изображения (макс. 50 МБ)
