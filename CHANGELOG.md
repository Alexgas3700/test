### Функциональность

#### Email Features
- Персонализация по имени, компании, email
- HTML-шаблоны с адаптивным дизайном
- Поддержка мобильных устройств
- Автоматическая генерация ссылок отписки
- Красивые кнопки призыва к действию (CTA)
- Профессиональные footer с контактами

#### Workflow Features
- Manual trigger для ручного запуска
- Поддержка Schedule trigger для автоматизации
- Обработка до 10 писем в партии (настраивается)
- Пауза 5 секунд между партиями (настраивается)
- Раздельная обработка успехов и ошибок
- Сводная статистика по кампании

#### Integration Features
- Google Sheets OAuth2 API
- SMTP поддержка (Gmail, SendGrid, Mailgun и др.)
- Автоматическое обновление таблиц
- Логирование в Google Sheets
- Email-отчеты администратору

### Метрики и аналитика

Workflow отслеживает:
- Общее количество обработанных контактов
- Количество успешно отправленных писем
- Количество ошибок
- Процент успешной доставки
- Временные метки отправки
- Детали ошибок

### Известные ограничения

- Gmail SMTP: до 500 писем/день (бесплатный аккаунт)
- SendGrid Free: до 100 писем/день
- Mailgun Free: до 5,000 писем/месяц
- Workflow обрабатывает письма последовательно (не параллельно)
- Требуется ручная настройка tracking для открытий и кликов

## Как читать этот changelog

### Типы изменений:
- `Добавлено` - новая функциональность
- `Изменено` - изменения в существующей функциональности
- `Устарело` - функциональность, которая скоро будет удалена
- `Удалено` - удаленная функциональность
- `Исправлено` - исправления багов
- `Безопасность` - изменения, связанные с безопасностью

### Версионирование:
- **MAJOR** (X.0.0) - несовместимые изменения API
- **MINOR** (0.X.0) - новая функциональность с обратной совместимостью
- **PATCH** (0.0.X) - исправления багов с обратной совместимостью
