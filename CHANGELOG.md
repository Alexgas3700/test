# Changelog

Все значимые изменения в проекте VK Content Workflow будут документированы в этом файле.

## [1.0.0] - 2026-02-07

### Добавлено

#### Workflows
- Базовый workflow для публикации контента в VK (`vk-content-workflow.json`)
  - Триггеры: webhook и schedule
  - Валидация контента
  - Публикация постов с текстом, изображениями, видео
  - Поддержка хэштегов и геолокации
  - Отложенная публикация
  - Обработка ошибок

- Расширенный workflow (`vk-content-advanced-workflow.json`)
  - AI-генерация контента через OpenAI
  - Автоматическая обработка изображений
  - Модерация контента
  - Расширенная аналитика
  - REST API ответы для webhook

#### VK API
- Максимум 3 запроса в секунду

- Максимум 10 вложений на пост
- Токен действителен ограниченное время

#### n8n
- Максимальный размер execution data
- Timeout для workflow (настраивается)

## Формат версий

Проект использует [Semantic Versioning](https://semver.org/):

- **MAJOR** версия - несовместимые изменения API
- **MINOR** версия - новая функциональность (обратно совместимая)
- **PATCH** версия - исправления ошибок (обратно совместимые)

## Типы изменений

- **Добавлено** - новая функциональность
- **Изменено** - изменения в существующей функциональности
- **Устарело** - функциональность, которая будет удалена
- **Удалено** - удаленная функциональность
- **Исправлено** - исправления ошибок
- **Безопасность** - исправления уязвимостей
