# VK Post Workflow - Сводка проекта

## Описание проекта

Комплексное решение для автоматической публикации постов с текстом и изображениями в VK (ВКонтакте) через n8n workflow automation platform.

## Структура проекта

### Workflow файлы

#### 1. `vk-post-workflow.json`
**Базовый workflow для публикации поста с одним изображением**

- ✅ Публикация текстовых постов
- ✅ Загрузка и прикрепление одного изображения
- ✅ OAuth2 аутентификация
- ✅ Простая структура из 5 узлов

**Узлы:**
1. Manual Trigger - запуск вручную
2. Set Post Data - настройка данных поста
3. Download Image - загрузка изображения по URL
4. Upload Photo to VK - загрузка фото на сервера VK
5. Create VK Post - создание поста

**Использование:** Идеально для начинающих и простых сценариев публикации.

---

#### 2. `vk-post-multiple-images-workflow.json`
**Продвинутый workflow для публикации поста с несколькими изображениями**

- ✅ Публикация с множественными изображениями (до 10)
- ✅ Автоматическая обработка массива URL
- ✅ Агрегация загруженных фото
- ✅ Форматирование attachments для VK API

**Узлы:**
1. Manual Trigger
2. Set Post Data - с массивом imageUrls
3. Prepare for Split - подготовка данных
4. Split Images - разделение массива изображений
5. Download Image - загрузка каждого изображения
6. Upload Photo to VK - загрузка на VK
7. Aggregate Photos - сбор всех загруженных фото
8. Format Attachments - форматирование для API
9. Create VK Post - публикация с несколькими фото

**Использование:** Для публикации фотоотчетов, галерей, коллекций изображений.

---

#### 3. `vk-scheduled-post-workflow.json`
**Workflow с автоматическим расписанием и обработкой ошибок**

- ✅ Автоматическая публикация по расписанию (cron)
- ✅ Получение контента из внешнего API
- ✅ Валидация данных перед публикацией
- ✅ Обработка ошибок с уведомлениями
- ✅ Telegram уведомления об успехе/ошибке

**Узлы:**
1. Schedule Trigger - запуск по расписанию (каждый день в 9:00)
2. Get Post Content - получение контента из API
3. Validate Content - проверка наличия данных
4. Download Image - загрузка изображения
5. Upload Photo to VK - загрузка на VK
6. Create VK Post - публикация
7. Set Success Response - формирование успешного ответа
8. Set Error Response - формирование ответа об ошибке
9. Send Success Notification - уведомление в Telegram об успехе
10. Send Error Notification - уведомление в Telegram об ошибке

**Использование:** Для автоматизированной публикации контента, контент-планов, регулярных постов.

## Технические характеристики

### Требования

- **n8n**: версия 0.200.0+
- **VK API**: версия 5.131+
- **Node.js**: 16+ (для n8n)

### Поддерживаемые операции

- `photos.upload` - загрузка фотографий
- `wall.post` - создание постов
- `groups.getById` - получение информации о группе
- `account.getAppPermissions` - проверка прав

### Лимиты

- **Запросы:** 3/сек (пользовательский токен), 20/сек (групповой)
- **Изображения:** до 10 в одном посте
- **Размер файла:** до 50 МБ
- **Текст:** до 16384 символов

## Безопасность

🔒 **Реализованные меры:**
- Безопасное хранение токенов в n8n credentials
- Поддержка OAuth2 с автоматическим обновлением
- Рекомендации по использованию environment variables
- Ограничение прав доступа (scope)

⚠️ **Рекомендации:**
- Не публикуйте токены в открытом доступе
- Используйте HTTPS для n8n сервера
- Регулярно обновляйте токены
- Мониторьте использование API
