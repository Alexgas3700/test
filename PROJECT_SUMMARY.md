## 🎯 Основные компоненты

### 1. Базовый Workflow (vk-content-workflow.json)

**Узлы:**
1. Schedule Trigger - запуск по расписанию
2. Generate Content - генерация контента из шаблонов
3. Validate Content - валидация перед публикацией
4. Check Validation - проверка результатов валидации
5. VK Post Content - публикация в VK через API
6. Check Success - проверка успешности публикации
7. Log Success - логирование успешных публикаций
8. Log Error - логирование ошибок
9. Get Post Stats - получение статистики (опционально)
10. Send Notification - отправка уведомлений (опционально)
11. Handle Validation Error - обработка ошибок валидации

**Возможности:**
- ✅ Автоматическая публикация каждые 6 часов
- ✅ 5 готовых шаблонов контента
- ✅ Валидация длины сообщения (до 4096 символов)
- ✅ Обработка ошибок VK API
- ✅ Логирование результатов

### 2. Продвинутый Workflow (vk-advanced-workflow.json)

**Дополнительные узлы:**
1. Select Topic - выбор темы для контента
2. OpenAI Generate Content - генерация через AI
3. Fallback Content - резервные шаблоны (8 тем)
4. Prepare VK Post - подготовка данных
5. Merge Results - объединение результатов

**Возможности:**
- ✅ Интеграция с OpenAI GPT-4
- ✅ 8 категорий контента с уникальными шаблонами
- ✅ Автоматический fallback при недоступности AI
- ✅ Расширенная валидация (включая запрещенные слова)
- ✅ Детальное логирование с метаданными

## 🗄️ База данных

### database-schema.sql

**Таблицы:**
1. `vk_posts_log` - логи всех публикаций
2. `vk_posts_stats` - статистика постов (просмотры, лайки, репосты)
3. `workflow_executions` - логи выполнения workflow
4. `content_templates` - шаблоны контента
5. `vk_rate_limits` - отслеживание rate limits

**Views:**
1. `vk_posts_analytics` - аналитика постов с engagement score
2. `workflow_performance` - производительность workflow

**Функции:**
- Автоматическое обновление `updated_at`
- Триггеры для таблиц

**Готовые данные:**
- 8 предзаполненных шаблонов контента

## 🚀 Как начать использовать

### Вариант 1: Быстрый старт (5 минут)
1. Читайте [QUICK_START.md](./QUICK_START.md)
2. Получите VK Access Token
3. Импортируйте `vk-content-workflow.json` в n8n
4. Настройте credentials и owner_id
5. Запустите тестовую публикацию

### Вариант 2: С AI генерацией (10 минут)
1. Следуйте Варианту 1
2. Получите OpenAI API ключ
3. Импортируйте `vk-advanced-workflow.json`
4. Настройте OpenAI credentials
5. Включите узел OpenAI Generate Content

### Вариант 3: Полная настройка (30 минут)
1. Следуйте Варианту 2
2. Создайте базу данных PostgreSQL
3. Выполните `database-schema.sql`
4. Включите узлы логирования
5. Настройте уведомления (Email/Telegram)
6. Настройте сбор статистики
