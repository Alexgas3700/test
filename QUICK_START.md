# Быстрый старт - VK Content Workflow для n8n

## За 5 минут к первой публикации!

### Шаг 1: Получите VK Access Token (2 минуты)

1. Перейдите на https://dev.vk.com/
2. Нажмите "Создать приложение"
3. Выберите тип "Standalone приложение"
4. Заполните название (например, "n8n VK Bot")
5. Скопируйте **App ID**

6. Откройте в браузере (замените YOUR_APP_ID):
```
https://oauth.vk.com/authorize?client_id=YOUR_APP_ID&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=wall,photos,groups&response_type=token&v=5.131
```

7. Разрешите доступ
8. Скопируйте **access_token** из URL (после `access_token=` и до `&`)

### Шаг 2: Узнайте ID вашей группы (1 минута)

1. Откройте вашу группу VK
2. Посмотрите на URL: `vk.com/club123456` или `vk.com/public123456`
3. Ваш ID группы: **-123456** (с минусом!)

Или для личной страницы:
- URL: `vk.com/id123456`
- ID: **123456** (без минуса)

### Шаг 3: Импортируйте workflow в n8n (1 минута)

1. Откройте n8n
2. Нажмите **"+"** → **"Import from File"**
3. Выберите файл `vk-content-workflow.json`
4. Нажмите **"Import"**

### Шаг 4: Настройте credentials (1 минута)

**Вариант A: Простой способ (рекомендуется для начала)**

1. Откройте узел **"VK Post Content"**
2. В параметре `access_token` замените `={{ $credentials.vkAccessToken }}` на ваш токен:
```
YOUR_ACTUAL_ACCESS_TOKEN_HERE
```

**Вариант B: Безопасный способ (рекомендуется для продакшена)**

1. В n8n перейдите в **"Credentials"** (левое меню)
2. Нажмите **"+ Add Credential"**
3. Выберите **"HTTP Request"**
4. Добавьте параметр:
   - Name: `vkAccessToken`
   - Value: `ваш_access_token`
5. Сохраните

### Шаг 5: Настройте ID группы (30 секунд)

1. Откройте узел **"Generate Content"**
2. Найдите строку:
```javascript
const owner_id = "-YOUR_GROUP_ID";
```
3. Замените на ваш ID:
```javascript
const owner_id = "-123456"; // Ваш ID группы
```

### Шаг 6: Тестовый запуск! (30 секунд)

1. Нажмите **"Test workflow"** в правом верхнем углу
2. Нажмите на узел **"Schedule Trigger"**
3. Нажмите **"Execute Node"**
4. Проверьте вашу группу VK - там должен появиться пост!

## Готово! 🎉

Ваш workflow работает! Теперь он будет автоматически публиковать контент каждые 6 часов.

## Настройка расписания

Откройте узел **"Schedule Trigger"**:

**Каждые N часов:**
```
Interval: Hours
Value: 4 (каждые 4 часа)
```

**В определенное время:**
```
Interval: Days
Time: 09:00 (каждый день в 9:00)
```

**По дням недели:**
```
Interval: Days
Days: Monday, Wednesday, Friday
Time: 12:00
```

## Кастомизация контента

### Изменить шаблоны сообщений

Откройте узел **"Generate Content"**, найдите:

```javascript
const contentTemplates = [
  "🚀 Новый день - новые возможности! #мотивация #успех",
  "💡 Совет дня: Начните с малого, но начните сегодня! #советы #развитие"
];
```

Добавьте свои сообщения:

```javascript
const contentTemplates = [
  "Ваш текст 1",
  "Ваш текст 2",
  "Ваш текст 3"
];
```

### Добавить изображение к посту

1. Загрузите изображение в VK
2. Скопируйте ID изображения (формат: `photo{owner_id}_{photo_id}`)
3. В узле **"Generate Content"** измените:

```javascript
const attachments = "photo-123456_789012"; // Ваше фото
```

## Troubleshooting

### ❌ "Invalid access token"
**Решение:** Получите новый токен (Шаг 1)

### ❌ "Access denied"
**Решение:** 
- Убедитесь, что вы администратор группы
- Проверьте права токена (должны быть: wall, photos, groups)

### ❌ "Owner id is invalid"
**Решение:** 
- Для группы ID должен быть с минусом: `-123456`
- Для личной страницы без минуса: `123456`

### ❌ Пост не появляется
**Решение:**
1. Проверьте логи n8n (нижняя панель)
2. Убедитесь, что группа публичная
3. Проверьте настройки приватности группы
